## Deterministic Script Toolbox

- `scripts/init_pkb.sh` for PKB skeleton initialization
- `scripts/gen_repo_map.sh` for repo-map tree refresh
- `scripts/gen_tech_stack.py` for manifest-based version extraction
- `scripts/gen_index.py` for `index.md` regeneration
- `scripts/check_pkb_staleness.py` for freshness checks
- `scripts/check_translation_sync.py` for `zh_CN` translation sync checks

- `scripts/extract_api_signatures.py` for Tauri command / REST signature extraction
- `scripts/gen_changelog.sh` for change-log drafts
- `scripts/gen_dep_graph.py` for static dependency Mermaid diagrams

## Translation and Build

```bash
cd [doc root]
poetry install
make html
```

If bilingual:

```bash
make gettext
make intl-update
make pkb-check-i18n
# or: python3 scripts/check_translation_sync.py --repo-root . --doc-dir .
```

## Suggested Checklist

1. Update the English source page first.
2. Run the relevant deterministic scripts so `_generated/` contains fresh sidecar artifacts.
3. Resolve or keep any `[NEEDS INPUT: ...]` markers deliberately; do not let the script invent rationale.
4. Refresh metadata footer.
5. Refresh translations if the English wording changed.
6. Rebuild the docs.
7. Verify links, diagrams, and navigation.
